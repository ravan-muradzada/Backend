### ‚úÖ 1. **Local** scope (inside a function)

```php
    $msg = "Hi";

```

### ‚úÖ 2. **Global** scope (outside functions)

```php

    // echo $name; ‚ùå Won‚Äôt work directly

```

---

## üîì `global` keyword

Use `global` inside a function to access global variables:

```php
$name = "Ravan";

function greet() {
    global $name;
    echo "Hello, $name";
}
greet(); // ‚úÖ Hello, Ravan
```

---

## üåê `$GLOBALS` superglobal

Another way to access global variables **without declaring** them:

```php
$name = "Ravan";

function greet() {
    echo "Hello, " . $GLOBALS['name'];
}
```

- `$GLOBALS` is an **array** holding all global variables.
    

---

## üß∑ `static` keyword (inside functions)

Keeps the variable value **between function calls** (doesn‚Äôt reset each time):

```php
function counter() {
    static $count = 0;
    $count++;
    echo $count . "\n";
}

counter(); // 1
counter(); // 2
counter(); // 3
```

- Without `static`, `$count` would reset to 0 every time.
